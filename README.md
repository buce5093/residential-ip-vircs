## 住宅IP是什么？为什么买住宅IP要谨慎

**住宅IP（Residential IP）**，是由真实互联网服务提供商（ISP）分配给家庭宽带用户的IP地址。它的核心特征是：在平台的风控系统眼里，它看起来就像一个真实用户在家里上网。

静态住宅IP则更进一步——它固定不变，长期绑定同一个地址，不会像动态IP那样频繁切换，特别适合账号养护、长期运营等需要稳定IP环境的场景。

简单说，同样的操作，用住宅IP做，平台以为你是普通用户；用机房IP做，平台知道你在走代理。

但市场上"住宅IP"这个词被滥用得很严重。从实际使用来看，以下几种情况都不是真正的住宅IP：

- 机房IP + 住宅IP标签（用检测工具一查就原形毕露）
- 共享宽带——IP是真的，但被几十人同时用，早就进黑名单了
- 动态住宅代理——每次连接IP都不一样，账号稳定性差
